## 🚀 Commands

### 🌐 Server
1. **Run server**:
```bash
sudo go run main.go s
```
- `s` – server mode.

2. **Generate client configuration**:
- In the interactive terminal, type:
```bash
gen
```

---

### 💻 Client
1. Copy the generated configuration file to:
```bash
settings/client/conf.json
```

2. **Start client**:
```bash
sudo go run main.go c
```
- `c` – client mode.
