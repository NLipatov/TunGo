# Quick start

## 1. Server: setup
<Tabs>
    <TabItem value="linux" label="Linux amd64" default>
        ```bash
        wget https://github.com/NLipatov/TunGo/releases/latest/download/tungo-linux-amd64 -O tungo
        chmod +x tungo
        sudo mv tungo /usr/local/bin/
        ```
    </TabItem>
</Tabs>

## 2. Server: generate client configuration

```bash
sudo tungo s gen
```

It will print out client configuration. Copy it to use on client.
