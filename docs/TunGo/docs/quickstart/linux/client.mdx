# Client
1. Download tungo:
<Download/>

2. Start the Client:
Run the tungo in your terminal to start the client:
```bash
sudo ./tungo
```

3. Add New Configuration:
In the interactive menu, add a new configuration and select it (for instance, name it nl-server).

4. Tunnel Established:
Once the configuration is selected, the tunnel is established and you are ready to go.
