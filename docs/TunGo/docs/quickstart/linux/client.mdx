# Client
1. Download tungo:
<Download/>

2. Save client configuration in `/etc/tungo/client_configuration.json` (it's generated on server's step 3)

3. Start the client
```bash
sudo ./tungo c
```

4. Type `exit` to stop tungo and deconfigure tunnel
