# TunGo: Worum geht es?

<p align="center">
  <img alt="TunGo running on a retro CRT terminal" src="/img/tungo-crt.svg" width="100%"/>
</p>

**TunGo** ist ein leichtgewichtiges VPN für moderne Anforderungen: **schnell**, **sicher** und **Open Source**.

### Hauptmerkmale:
- 🧩 **Einfach zu deployen und konfigurieren**.
- 🚀 **Hohe Leistung** — nahezu keine Allokationen auf dem Hot Path (benchmarked).
- 📦 **Geringer Speicherbedarf** — ≈5–15 MB **RSS** unter Last, ≈5–8 MB im Leerlauf.
- 🔒 **Ende-zu-Ende-Verschlüsselung** — Noise IK-Handshake für gegenseitige Authentifizierung; X25519 (Curve25519 ECDH) für Schlüsselaustausch; ChaCha20-Poly1305 (AEAD) für Traffic-Verschlüsselung.
- ⚡ **Von Grund auf neu gebaut** — kein Legacy, kein Bloat. Sauberer, lesbarer Go-Code.
- 🌐 **IoT- und Embedded-ready** — optimiert für kleine Geräte und ressourcenbeschränkte Umgebungen.
- 🛡️ **Open Source** — AGPL-3.0-only; kommerzielle Lizenzen verfügbar.
