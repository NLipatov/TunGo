# TunGo とは？

<p align="center">
  <img alt="Two gophers dancing tango" src="https://i.ibb.co/GvSCJ01n/Tun-Go-logo.png" width="40%"/>
</p>

**TunGo** は現代のニーズに合わせて設計された軽量 VPN です：**高速**、**セキュア**、**オープンソース**。

### 主な特徴：
- 🧩 **デプロイと設定が簡単**。
- 🚀 **ハイパフォーマンス** — ホットパスでほぼゼロのメモリ割り当て（ベンチマーク済み）。
- 📦 **小さなメモリフットプリント** — 負荷時 ≈5–15 MB **RSS**、アイドル時 ≈5–8 MB。
- 🔒 **エンドツーエンド暗号化** — X25519（Curve25519 ECDH）で鍵交換、ChaCha20-Poly1305（AEAD）でトラフィック暗号化、Ed25519 で認証。
- ⚡ **スクラッチから構築** — レガシーなし、ブロートなし。クリーンで読みやすい Go コード。
- 🌐 **IoT・組み込み対応** — 小型デバイスやリソース制限のある環境向けに最適化。
- 🛡️ **オープンソース** — AGPL-3.0-only、商用ライセンスも利用可能。
