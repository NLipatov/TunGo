# Docker での TunGo サーバー

**1️⃣ イメージを準備**
- 🐳 公式：
```bash
  docker pull nlipatov/tungo:latest
````

* 🛠 または自分でビルド：

  ```bash
  cd src/
  docker buildx build -t nlipatov/tungo:latest .
  ```

**2️⃣ コンテナを実行**

```bash
docker run -d \
  --name tungo \
  --restart always \
  --network host \
  --device /dev/net/tun \
  --cap-add NET_ADMIN \
  -e EnableUDP=true \
  -e EnableTCP=false \
  -v tungo_volume:/etc/tungo \
  nlipatov/tungo:latest
```

**3️⃣ クライアント設定を生成**

```bash
docker exec tungo bash -c "tungo s gen"
```

> コンテナ名の代わりにコンテナ ID を使用することもできます。

📖 次のステップについては[クライアントセットアップ](/docs/QuickStart#%EF%B8%8F-client-setup)を参照してください。
