# TunGo: Do que se trata?

<p align="center">
  <img alt="TunGo em execução em um terminal CRT retrô" src="/img/tungo-crt.svg" width="100%"/>
</p>

**TunGo** é uma VPN leve projetada para necessidades modernas: **rápida**, **segura** e **de código aberto**.

### Características principais:
- 🧩 **Fácil de implantar e configurar**.
- 🚀 **Alto desempenho** — quase zero alocações no caminho crítico (benchmarked).
- 📦 **Pequena pegada de memória** — ≈5–15 MB **RSS** sob carga, ≈5–8 MB em repouso.
- 🔒 **Criptografia de ponta a ponta** — handshake Noise IK para autenticação mútua; X25519 (Curve25519 ECDH) para acordo de chaves; ChaCha20-Poly1305 (AEAD) para criptografia de tráfego.
- ⚡ **Construído do zero** — sem legado, sem inchaço. Código Go limpo e legível.
- 🌐 **Pronto para IoT e embarcados** — otimizado para dispositivos pequenos e ambientes com recursos limitados.
- 🛡️ **Código aberto** — AGPL-3.0-only; licenças comerciais disponíveis.
