# TunGo: Apa Itu?

<p align="center">
  <img alt="TunGo running on a retro CRT terminal" src="/img/tungo-crt.svg" width="100%"/>
</p>

**TunGo** adalah VPN ringan yang dirancang untuk kebutuhan modern: **cepat**, **aman**, dan **open-source**.

### Fitur utama:
- 🧩 **Mudah di-deploy dan dikonfigurasi**.
- 🚀 **Performa tinggi** — hampir nol alokasi di jalur kritis (sudah di-benchmark).
- 📦 **Jejak memori kecil** — ≈5–15 MB **RSS** saat beban, ≈5–8 MB saat idle.
- 🔒 **Enkripsi end-to-end** — handshake Noise IK untuk autentikasi timbal balik; X25519 (Curve25519 ECDH) untuk pertukaran kunci; ChaCha20-Poly1305 (AEAD) untuk enkripsi lalu lintas.
- ⚡ **Dibangun dari nol** — tanpa kode lawas, tanpa bloat. Kode Go yang bersih dan mudah dibaca.
- 🌐 **Siap untuk IoT & Embedded** — dioptimalkan untuk perangkat kecil dan lingkungan dengan sumber daya terbatas.
- 🛡️ **Open source** — AGPL-3.0-only; lisensi komersial tersedia.
