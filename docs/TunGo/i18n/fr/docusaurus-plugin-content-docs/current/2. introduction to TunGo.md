# TunGo : De quoi s'agit-il ?

<p align="center">
  <img alt="TunGo running on a retro CRT terminal" src="/img/tungo-crt.svg" width="100%"/>
</p>

**TunGo** est un VPN léger conçu pour les besoins modernes : **rapide**, **sécurisé** et **open-source**.

### Caractéristiques principales :
- 🧩 **Facile à déployer et configurer**.
- 🚀 **Haute performance** — quasi zéro allocations sur le chemin critique (benchmarké).
- 📦 **Faible empreinte mémoire** — ≈5–15 Mo **RSS** sous charge, ≈5–8 Mo au repos.
- 🔒 **Chiffrement de bout en bout** — handshake Noise IK pour l'authentification mutuelle ; X25519 (Curve25519 ECDH) pour l'échange de clés ; ChaCha20-Poly1305 (AEAD) pour le chiffrement du trafic.
- ⚡ **Construit de zéro** — sans héritage, sans bloat. Code Go propre et lisible.
- 🌐 **Prêt pour l'IoT et l'embarqué** — optimisé pour les petits appareils et les environnements contraints.
- 🛡️ **Open source** — AGPL-3.0-only ; licences commerciales disponibles.
