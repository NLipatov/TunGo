# 🚀 Démarrage Rapide

Mettez en place votre serveur et client TunGo en quelques minutes.

---

## 🖥️ Configuration du Serveur

1️⃣ **Télécharger et installer**
   <Tabs>
     <TabItem value="linux-server-amd64" label="Linux amd64" default>
     ```bash
     wget https://github.com/NLipatov/TunGo/releases/latest/download/tungo-linux-amd64 -O tungo && \
     chmod +x tungo && \
     sudo mv tungo /usr/local/bin/
     ```
     </TabItem>
     <TabItem value="linux-server-arm64" label="Linux arm64">
     ```bash
     wget https://github.com/NLipatov/TunGo/releases/latest/download/tungo-linux-arm64 -O tungo && \
     chmod +x tungo && \
     sudo mv tungo /usr/local/bin/
     ```
     </TabItem>
   </Tabs>

2️⃣ **Générer la configuration client**
   ```bash
   sudo tungo s gen
   ```
   > Copiez la configuration affichée pour votre client.

3️⃣ **Démarrer le serveur**
   ```bash
   sudo tungo
   ```

:::tip Daemon systemd du serveur (Linux)
Consultez <a href="/docs/Advanced/Linux/Setup server systemd unit">ce guide</a> pour exécuter TunGo en tant que service systemd (démarrage automatique/redémarrage au boot).
:::
