# TunGo: ¿De qué se trata?

<p align="center">
  <img alt="TunGo running on a retro CRT terminal" src="/img/tungo-crt.svg" width="100%"/>
</p>

**TunGo** es una VPN ligera diseñada para necesidades modernas: **rápida**, **segura** y **de código abierto**.

### Características principales:
- 🧩 **Fácil de desplegar y configurar**.
- 🚀 **Alto rendimiento** — casi cero asignaciones en la ruta crítica (probado con benchmarks).
- 📦 **Huella de memoria mínima** — ≈5–15 MB **RSS** bajo carga, ≈5–8 MB en reposo.
- 🔒 **Cifrado de extremo a extremo** — handshake Noise IK para autenticación mutua; X25519 (Curve25519 ECDH) para acuerdo de claves; ChaCha20-Poly1305 (AEAD) para cifrado de tráfico.
- ⚡ **Construido desde cero** — sin legado, sin bloat. Código Go limpio y legible.
- 🌐 **Listo para IoT y embebidos** — optimizado para dispositivos pequeños y entornos con recursos limitados.
- 🛡️ **Código abierto** — AGPL-3.0-only; licencias comerciales disponibles.
