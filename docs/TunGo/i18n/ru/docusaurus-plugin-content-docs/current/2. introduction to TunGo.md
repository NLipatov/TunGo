# TunGo: Что это?

<p align="center">
  <img alt="TunGo, запущенный на ретро-терминале с ЭЛТ" src="/img/tungo-crt.svg" width="100%"/>
</p>

**TunGo** — это легковесный VPN, разработанный для современных задач: **быстрый**, **безопасный** и **с открытым исходным кодом**.

### Ключевые особенности:
- 🧩 **Простое развёртывание и настройка**.
- 🚀 **Высокая производительность** — почти нулевые аллокации на горячем пути (проверено бенчмарками).
- 📦 **Минимальный расход памяти** — ≈5–15 МБ **RSS** под нагрузкой, ≈5–8 МБ в покое.
- 🔒 **Сквозное шифрование** — хендшейк Noise IK для взаимной аутентификации; X25519 (Curve25519 ECDH) для обмена ключами; ChaCha20-Poly1305 (AEAD) для шифрования трафика.
- ⚡ **Написан с нуля** — без legacy-кода, без раздувания. Чистый, читаемый код на Go.
- 🌐 **Готов для IoT и встраиваемых систем** — оптимизирован для маломощных устройств и ограниченных сред.
- 🛡️ **Открытый исходный код** — AGPL-3.0-only; доступны коммерческие лицензии.
