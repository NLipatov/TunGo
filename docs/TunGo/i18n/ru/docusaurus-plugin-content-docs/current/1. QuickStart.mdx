# 🚀 Быстрый старт

Запустите сервер и клиент TunGo за несколько минут.

---

## 🖥️ Настройка сервера

1️⃣ **Скачать и установить**
   <Tabs>
     <TabItem value="linux" label="Linux amd64" default>
     ```bash
     wget https://github.com/NLipatov/TunGo/releases/latest/download/tungo-linux-amd64 -O tungo && \
     sudo install -o root -g root -m 0755 tungo /usr/local/bin/tungo
     ```
     </TabItem>
     <TabItem value="linux-arm64" label="Linux arm64">
     ```bash
     wget https://github.com/NLipatov/TunGo/releases/latest/download/tungo-linux-arm64 -O tungo && \
     sudo install -o root -g root -m 0755 tungo /usr/local/bin/tungo
     ```
     </TabItem>
   </Tabs>

2️⃣ **Сгенерировать конфигурацию клиента**
   ```bash
   sudo tungo s gen
   ```
   > Скопируйте выведенную конфигурацию для вашего клиента.

3️⃣ **Запустить сервер**
   ```bash
   sudo tungo
   ```

:::tip Systemd-демон сервера (Linux)
Смотрите <a href="/docs/Advanced/Linux/Setup server systemd unit">это руководство</a> для запуска TunGo как сервиса systemd (автозапуск/перезапуск при загрузке).
:::
