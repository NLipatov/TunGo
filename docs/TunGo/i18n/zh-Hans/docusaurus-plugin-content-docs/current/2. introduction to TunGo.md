# TunGo：简介

<p align="center">
  <img alt="Two gophers dancing tango" src="https://i.ibb.co/GvSCJ01n/Tun-Go-logo.png" width="40%"/>
</p>

**TunGo** 是一款为现代需求设计的轻量级 VPN：**快速**、**安全**、**开源**。

### 主要特性：
- 🧩 **易于部署和配置**。
- 🚀 **高性能** — 热路径上几乎零内存分配（已基准测试）。
- 📦 **极小内存占用** — 负载下约 5–15 MB **RSS**，空闲时约 5–8 MB。
- 🔒 **端到端加密** — X25519（Curve25519 ECDH）用于密钥协商；ChaCha20-Poly1305（AEAD）用于流量加密；Ed25519 用于身份认证。
- ⚡ **从零构建** — 无遗留代码，无臃肿。简洁、可读的 Go 代码。
- 🌐 **物联网和嵌入式就绪** — 针对小型设备和资源受限环境优化。
- 🛡️ **开源** — AGPL-3.0-only 许可；可提供商业许可。
